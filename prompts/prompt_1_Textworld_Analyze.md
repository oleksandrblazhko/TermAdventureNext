# Person
Ти спеціаліст з аналізу json-файлів.
Є проект Textworld - https://textworld.readthedocs.io/en/stable/
SourceCode проекту - https://github.com/Microsoft/TextWorld
Команда tw-make створює декілька файлів для текстового квесту:
tw-make tw-simple --goal brief --seed 5 --output simple_game.z8 --rewards sparse
Одний із результуючих файлів є json-файл, який містить текстовий опис гри.

# Task
1) проаналізуй проект Textworld
2) визнач алгоритм роботи команди tw-make
3) визнач всі можливі сутності гри
4) визнач ієрархію між сутностями
5) для кожної сутності визнач всі можливі стани сутностів, в яких сутності можуть перебути
6) для кожної сутності гри визнач всі можливі дії над сутностями
7) прочитай всі приклади json-файлів, які було створено командою tw-make
8) представ опис сутностей та опис дій у вигляді таблиці опису гри:
- id ієрархії, наприклад, 1., 1.1., 1.2, 2., 2.1, 2.2 ...
- parent-сутність
- child-сутність
- назва дії гравця (назва можливої дії гравця по відношенню до child-сутності)
- опис дії
- приклад дії (із прикладів json-файлів)

# Context
1) приклади json-файлів:
- prompts\test_game_1.json
- prompts\test_game_2.json
- prompts\test_game_3.json

# Format
1) в таблиці назви parent-сутність, child-сутність та назва дії надавай оригінальною англійською, а в дужках - українською
2) в таблиці - опис дії, приклад дії надавай українською
3) файл збереження результатів - promts\TW-simple_Analyze_gem.md
