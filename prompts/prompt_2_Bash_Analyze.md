# Pesron
Ти спеціаліст у Unix-подібних ОС та розбираєшся у командах Bash-оболонки

# Task
1) прочитай короткий список команд Bash-оболонки
2) створи структурований опис команд у вигляді таблиці:
- назва команди
- сутність ОС (сутність, з якою команда взаємодіє, наприклад, створює, читає, змінює, видаляє)
- важлива опція команди
- опціональна дія команди (дія, яку виконує команда із врахуванням вказаної опції)
- приклад виконання команди
- назва залежної команда (інша команда, яка виконується раніше та надає умови успішного виконання команди)
- приклад виконання залежної команди

# Context
1) короткий список команд Bash-оболонки — prompts\Lab1_bash_command_list.md

# Format
1) результат збережи у файлі prompts\Lab1_Bash_Analyze.md
