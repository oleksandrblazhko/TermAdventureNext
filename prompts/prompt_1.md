# Context
Список команд Bash-оболонки:
1) uname – надрукувати інформацію про комп’ютер, використовуючи прапорці: n – мережева назва комп’ютера, m – апаратна платформа комп’ютера, s – назва ядра ОС, a – повний опис.
2) pwd – переглянути повний шлях від кореневого каталогу до поточного робочого каталогу
3) whoami - переглянути ім'я користувача, який запустив оболонку
4) date	- переглянути поточну дату із визначенням окремих показників часу, наприклад, команда:
date +'%e день %m місяця %G року' – виведе день (%e), місяць (%m) та рік (%G), але більше форматів можна отримати через виклик date –help
5) echo - $змінна_ середовища - переглянути значення змінної оточуючого середовища ОС, першим прикладом, якої є змінна $PATH із переліком каталогів, необхідних для швидкого пошуку файлів, які можуть виконуватися в командному рядку без вказування повного шляху до файлу, при цьому каталоги розділяються символом двокрапка ‘:’
6) ls каталог (list – список) - переглянути вміст каталогу за вказаною назвою або поточного каталогу, якщо назва не вказана
7) less файл - переглянути вміст файлу (для завершення перегляду файлу використовується клавіша q ).
8) cd каталог - перейти до іншого каталогу або домашнього каталогу користувача, якщо назва каталогу не вказана: 
 для переходу до каталогу на рівень вище (батьківський каталог) у назві каталогу вказуються символи дві крапки .. , наприклад, cd ..
9) pushd каталог - перейти до іншого каталогу за вказаною назвою з можливістю швидкого повернення до нього, завдяки тому, що команда формує власний стек із назвами каталогів та виводить його на екран; виконання команди без параметру перемикає між двома верхніми каталогами у стеку
10) popd - повернутися до каталогу, з якого виконано переміщення після останньої команди pushd, взявши назву каталогу з вершини стеку
