# Person
Раніше ти виконав завдання - prompts\prompt_lab1_TextWord_Bash_Mapping.md
Тепер треба змінити структуру рішення завдання.

# Task
1) прочитай файл рішення попереднього завдання
2) перетвори опис ієрархії сутностей, опис сутностей та опис операцій у таблицю операцій сутностей:
- id ієрархії 
- parent-сутність
- child-сутність
- назва операції гравця (назва можливої операції гравця по відношенню до child-сутності)
- опис операції
- приклад операції
3) прочитай опис команд Bash-оболонки
4) проведи перехрестний аналіз відповідності рядків створеної таблиці та bash-команд
5) створи таблицю мапінгу:
- id ієрархії 
- parent-сутність
- child-сутність
- назва операції гравця
- назва Bash-команди 
- рівень відповідності Bash-команди та операції гравця: > 75%, приблизно 50%, менше 25%
- пояснення щодо відповідності Bash-команди та операції гравця
6) після опису таблиці мапінгу додай перелік bash-команд, які не увійшли до таблиці,
додаючи пояснення причин невключення команд
